Detailed Description

Ray class which is a representation of a ray function.

The Ray class represents a path that the light travels in a virtual scene. To find an object that ray intersects with we have to determine the ray function:

\[ p(t) = A + t*B \]

where p is a 3D position along a line in 3D. A is the ray origin, B is the ray direction and the ray parameter t is a real number. Depending on the t point p(t) moves along the ray. For example for t=2 and t=0 the ray funtions would be:

\[ p(2) = A + 2*B, \\ p(0) = A + 0*B => p(0) = A \]
