\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_light_intensity}{Light\+Intensity}} class that represents light intensiti for a color in R\+GB colorspace. 

This class represents ambient light intensity in R\+GB colorspace. It has no position, and it\textquotesingle{}s the same for every object in a virtual scene.
