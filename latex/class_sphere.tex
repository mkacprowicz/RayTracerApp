\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_sphere}{Sphere}} class which is a representation of a sphere solid. 

The {\ttfamily \mbox{\hyperlink{class_sphere}{Sphere}}} class is a representation of a Set of points that are equaly apart (sphere radius) in a 3D space. In order to define a sphere, we need to determine it\textquotesingle{}s center, with a Point or \mbox{\hyperlink{class_vector}{Vector}}, and it\textquotesingle{}s radius -\/ real number. \mbox{\hyperlink{class_sphere}{Sphere}} that has a center equal to $ c = (c_x, c_y, c_z) $ and radius equal to $ R $ can be determied with a formula\+: \[ (x-c_x)^2 + (y-c_y)^2 + (z-c_z)^2 - R^2 = 0 \] which is equivalent to a vector formula\+: \[ (p-c) \cdot (p-c) - R^2 = 0 \] Every point that satisfies that equation belongs to the sphere.

{\footnotesize\ttfamily bool Sphere\+::\+Hit\+Test (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{class_ray}{Ray}}}]{ray,  }\item[{float \&}]{distance,  }\item[{\mbox{\hyperlink{class_vector}{Vector}} \&}]{normal }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [virtual]}}

Function that checks whether or not \mbox{\hyperlink{class_sphere}{Sphere}} intersects with a \mbox{\hyperlink{class_ray}{Ray}}. There are three possible cases that can occur\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_ray}{Ray}} doesn\textquotesingle{}t hit a \mbox{\hyperlink{class_sphere}{Sphere}}
\item \mbox{\hyperlink{class_ray}{Ray}} hits a \mbox{\hyperlink{class_sphere}{Sphere}} in one point
\item \mbox{\hyperlink{class_ray}{Ray}} hits a \mbox{\hyperlink{class_sphere}{Sphere}} in two points To examine which case is valid we have to determine an equation\+: \[ ||A + Bt -c||^2 = r^2 <=> (A + Bt - c) \cdot (A + Bt - c) = r^2 \\ t^2(B \cdot B) + 2t(B \cdot (A - c)) + (A - c) \cdot (A - c) = r^2 \\ t^2(B \cdot B) + 2t(B \cdot (A - c)) + (A - c) \cdot (A - c) - r^2 = 0 \\ \] which is equal to quadratic equation\+: \[ at^2 + bt + c = 0 \] where\+: \[ a = B \cdot B = ||B||^2 \\ b = 2(B \cdot (A - c)) \\ c = (A - c) \cdot (A - c) - r^2 = ||A - c||^2 - r^2 \] what can be simplified\+: \[ t = \frac{-2(B \cdot (A - c)) \pm \sqrt{(2(B \cdot (A - c)))^2 - 4||B||^2 (||A - c||^2 - r^2)}}{2||B||} \\ ||A||^2 = 1, so \\ t = -(B * (A - c)) \pm \sqrt{(B \cdot (A - c))^2 - ||A - c||^2 + r^2)} \]
\item if the sqrt value $ (B \cdot (A - c))^2 - ||A - c||^2 + r^2) < 0$ equation cannot be solved, thus \mbox{\hyperlink{class_ray}{Ray}} is not intersecting with a \mbox{\hyperlink{class_sphere}{Sphere}}
\item if it\textquotesingle{}s equal to $ 0 $ there is only one solution -\/ the \mbox{\hyperlink{class_ray}{Ray}} intersects with a \mbox{\hyperlink{class_sphere}{Sphere}} in one spot
\item if it\textquotesingle{}s more than zero the \mbox{\hyperlink{class_ray}{Ray}} intersects with the Spehere in two spots
