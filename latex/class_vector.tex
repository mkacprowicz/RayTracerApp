\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_vector}{Vector}} class which is a representation of a 3D vector. 

The {\ttfamily \mbox{\hyperlink{class_vector}{Vector}}} class is a representation of a geometric object that has magnitude (or length) and direction. Vectors can be added to other vectors according to vector algebra.
