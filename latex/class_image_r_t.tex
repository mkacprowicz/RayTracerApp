\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_image_r_t}{Image\+RT}} class which is a representation of a image. 

The {\ttfamily \mbox{\hyperlink{class_image_r_t}{Image\+RT}}} class represents an image to render. Contains image height and width, vector with image pixels and number of samples to sampling.
